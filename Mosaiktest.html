Bilder bauen

Was ist Ihre Aufgabe?

Auf dem Bildschirm sehen Sie gleich rechts unten ein Bild.
Ihre Aufgabe ist es dieses Bild aus seinen Einzelbausteinen nachzubauen.

Wählen Sie zuerst einen Baustein, den Sie setzen wollen, aus. Danach wählen Sie ein leeres Feld in Ihrem „Baufeld“, in das Sie den Baustein legen wollen.
Wenn Sie ein richtiges Feld ausgewählt haben, wird der Baustein hier eingesetzt. Wenn Sie ein falsches Feld auswählen, erscheint es rot – dann können Sie sich entweder für ein anderes Feld entscheiden oder zuerst einen anderen Baustein wählen.

Versuchen Sie so schnell und so fehlerfrei wie möglich zu arbeiten!
